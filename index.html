Glossary

Played - The number of games played on the given map.

First Ban - The number of times the given map was banned with one of the first 2 bans.

Ban - The number of times the given map was banned (excluding the first 2 bans).

First Pick - The number of times the given map was picked with one of the first 2 picks.

Pick - The number of times the given map was picked (exluding the first 2 picks and the remaining pick).

Remaining - The number of times the given map was the final map to be picked.

First Ban% - The percentage of map drafts where the given map was a first ban.

Ban% - The percentage of map drafts where the given map was banned (excluding the first 2 bans).

First Pick% - The percentage of map drafts where the given map was a first pick.

Pick% - The percentage of map drafts where the given map was picked (exluding the remaining pick).

Remaining% - The percentage of map drafts where the given map was the final map to be picked.

Wins - The number of times the player who picked the given map won the game (excluding the remaining pick).

Win% - The percentage of times the player who picked the given map won the game (excluding the remaining pick).
